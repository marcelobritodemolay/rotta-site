  1. Na etapa de setup, ative A/B Test
  2. Insira a Linha A e a Linha B (veja cada template acima)
  3. Configuracao recomendada:
    • Tamanho do teste: 20% da lista (10% recebe A, 10% recebe B)
    • Criterio vencedor: Open Rate (taxa de abertura)
    • Tempo de teste: 2 horas
    • Apos o teste: enviar a versao vencedora pros 80% restantes
Dica: Se a lista tiver menos de 100 contatos, desative o A/B test e alterne manualmente entre as linhas A e B a cada semana. O teste so faz sentido estatisticamente com volume.

Passo 4 — Selecionar a Lista de Envio

  1. Na etapa Recipients, clique em Select a list
  2. Escolha a lista correspondente (ex: Quiz Carolina #3)
  3. NAO marque "Send to all contacts" — envie apenas para a lista especifica
  4. Se quiser excluir quem ja comprou, crie uma lista de exclusao Clientes Ativos

Passo 5 — Inserir o Template HTML

  1. Na etapa Design, escolha Paste your code (colar codigo)
  2. Copie o HTML completo do template correspondente (use os botoes "Copiar HTML" acima)
  3. Cole no editor de codigo do Brevo
  4. Clique em Preview para verificar como ficara no desktop e mobile
  5. Substitua todas as variaveis {{exemplo}} pelo conteudo real da semana
Variaveis do Brevo: O template ja usa {{ unsubscribe }} e {{ organization.name }} — essas sao variaveis nativas do Brevo e serao preenchidas automaticamente. NAO altere essas.

Passo 6 — Agendar Envio

  1. Na etapa Schedule, selecione Schedule for later
  2. Escolha: Domingo, 19:00, fuso BRT (America/Sao_Paulo)
  3. Revise tudo e clique em Schedule
Importante: O Brevo NAO tem envio recorrente nativo em campanhas. Voce precisara criar uma nova campanha toda semana (ou duplicar a anterior). Para automatizar, veja o Passo 7.

Passo 7 — Alternativa: Automacao para Recorrencia

Se quiser automatizar o envio semanal sem criar campanha toda semana:

  1. Acesse Automation > Create a workflow
  2. Trigger: Contact is in list — selecione a lista correspondente
  3. Adicione uma condicao de tempo: Wait until > Day of week: Sunday, Time: 19:00 BRT
  4. Acao: Send email — insira o template
  5. Adicione um Wait de 7 dias apos o envio
  6. Use um Loop para voltar ao inicio (assim o contato recebe toda semana)
Recomendacao: Para as primeiras 4-6 semanas, use Campanhas manuais. Isso permite ajustar o conteudo e testar o que funciona. Depois que tiver um ritmo, migre para Automacao.

Passo 8 — Checklist Semanal (todo sabado)

Execute este checklist para preparar as 5 newsletters de domingo:

  • Escolher destino/experiencia da semana para cada persona
  • Escrever copy personalizada (substituir variaveis {{}})
  • Selecionar/produzir imagens (600x220px para hero, 260x140px para thumbs)
  • Subir imagens no Brevo (Media Library) e copiar URLs
  • Substituir placeholders de imagem pelos URLs reais
  • Inserir links do Instagram (carrossel/reel da semana)
  • Duplicar campanha anterior de cada persona
  • Atualizar conteudo + linhas de assunto
  • Testar preview desktop + mobile
  • Enviar email teste para si mesmo
  • Agendar para domingo 19h BRT

Passo 10 — Dicas de Deliverability

  • Autenticacao: Configure SPF, DKIM e DMARC no dominio rottacuradoria.com (Brevo orienta em Settings > Senders)
  • Remetente: Use sempre o mesmo endereco de email para construir reputacao
  • Background claro: Os templates ja usam fundo #F2EDE8 (creme claro) — isso melhora a entrega e evita filtros de spam
  • Proporcao texto/imagem: Mantenha pelo menos 60% texto e 40% imagem
  • Link de descadastro: Sempre visivel no footer (ja incluso nos templates)
  • Aquecimento: Se o dominio e novo, comece enviando para listas menores e aumente gradualmente
